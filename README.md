# Como usar/Descrição
1. É um site de cadastro de Projetos
2. Nele, você tem uma aba de dashboard com cards mostrando o total de projetos cadastrados,
os projetos que estão em andamento, os que estão concluídos e os que estão pendentes.
Além de um botão de "Gerenciar Projetos", que redireciona para a aba de projetos em si.
3. Nessa aba de projetos, temos todos os projetos organizados em forma de lista, com a opção
de criar um novo projeto, editar ou excluir.
# Login
1. É feito pelo google normalmente.   
# Banco
1. Usei o Supabase.
# Como Rodar
1. Backend: cd backend, npm install, npm run dev.
2. Frontend: cd frontend, npm install, npm run dev.
3. Será executado em http://localhost:5173
